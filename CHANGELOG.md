# 📝 История изменений

Все значимые изменения в проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [2.0.0] - 2026-03-13

### ✨ Добавлено

#### Ядро системы
- **Core\Config** - Централизованная конфигурация через .env
- **Core\Database** - Улучшенная PDO обёртка с поддержкой транзакций
- **Core\Security** - CSRF, XSS защита, валидация файлов
- **Core\Session** - Улучшенное управление сессиями с flash-сообщениями
- **Core\Logger** - Многоуровневое логирование (debug, info, warning, error)
- **Core\Validator** - Мощная система валидации с 20+ правилами

#### Безопасность
- CSRF токены для всех форм
- XSS фильтрация всех выводов
- Валидация загружаемых файлов
- Безопасные сессии (httponly, secure)
- Rate limiting для попыток входа

#### Логирование
- Логи по датам (YYYY-MM-DD.log)
- Контекстное логирование с метаданными
- Автоматическая ротация логов
- Логирование PHP ошибок в production

#### Глобальные хелперы
```php
db()                    // База данных
config($key)            // Конфигурация
e($value)               // XSS защита
csrf_token()            // CSRF токен
csrf_field()            // HTML поле с CSRF
redirect($url)          // Редирект
back()                  // Назад
post($key)              // Получение POST
get($key)               // Получение GET
flash($key, $value)     // Flash сообщения
logger()                // Логгер
is_development()        // Проверка окружения
is_production()         // Проверка production
format_date($date)      // Форматирование даты
format_datetime($dt)    // Форматирование даты-времени
```

#### Документация
- Подробный README.md с примерами
- MIGRATION_GUIDE.md для миграции со v1.0
- SECURITY.md с рекомендациями
- .env.example с комментариями

### 🔄 Изменено

- **config.php** - Полностью переписан с использованием новых классов
- **Обратная совместимость** - старые функции помечены @deprecated, но работают
- **Автозагрузчик** - Добавлен SPL autoloader для классов

### 🚀 Улучшено

- **Производительность** - Оптимизированы запросы к БД
- **Кэширование** - Улучшенное управление кэшем
- **Сессии** - Более безопасная работа с сессиями
- **Обработка ошибок** - Улучшенное логирование ошибок

### 🛡️ Безопасность

- Исправлены потенциальные XSS уязвимости
- Добавлена защита от CSRF атак
- Улучшена валидация файлов
- Добавлен rate limiting

### 📝 Примечания по миграции

Для обновления с v1.0:
1. Сделайте резервную копию
2. Обновите файлы
3. Создайте .env из .env.example
4. Создайте директорию logs/
5. Проверьте права доступа

Подробнее: [MIGRATION_GUIDE.md](MIGRATION_GUIDE.md)
